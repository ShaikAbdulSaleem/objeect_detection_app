⚠️ Live Webcam Detection

Direct live webcam detection and server-side processing for real-time video streams are complex to set up in typical hosted environments like Render without dedicated streaming infrastructure (e.g., WebSockets, RTMP, or client-side processing).

For this application, live detection would involve:

  • Capturing frames from the webcam in your browser using JavaScript.
  • Sending these frames (e.g., as base64 encoded images) to a backend API endpoint.
  • The backend processes the image and sends back the annotated image.
  • The browser displays the annotated image.

This adds significant complexity to the project. For a simple initial deployment, it's recommended to focus on image and video file uploads.
